# umi project

## Getting Started

Install dependencies,

```bash
$ yarn
```

Start the dev server,

```bash
$ yarn start
```
## 设备出租平台
`这是个外包项目，包含设备出租前台，零件售卖，二手设备出租，设备维修，课程售卖，相关行业新闻等模块的前台展示和后台管理`
## 平台
* PC
  1. 前台门户(买家和卖家)
  2. 后台管理平台
* H5
  1. 前台门户
* 小程序
